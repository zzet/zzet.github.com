## Введение в Sublime Merge

Sublime Merge — это быстрый и мощный Git клиент для операционных систем Windows, macOS и Linux, созданный командой разработчиков Sublime HQ, знаменитой своим текстовым редактором Sublime Text. Если вы когда-то использовали Sublime Text, то заметите знакомый дизайн и философию — минимализм, скорость и мощь.

Sublime Merge не является полностью бесплатным инструментом, но предлагает длительный пробный период и доступную цену. Главное отличие Sublime Merge от других Git клиентов — это потрясающая скорость работы, особенно при работе с большими репозиториями с тысячами коммитов, и один из лучших встроенных diff-просмотрщиков.

Инструмент идеально подходит для разработчиков, которые уже используют Sublime Text, или для тех, кто просто ценит скорость и надёжность. В отличие от {{< relref "lazygit" >}}, Sublime Merge предоставляет полноценный графический интерфейс, но без излишней перегруженности, как в GitHub Desktop.

## Установка и лицензирование

### Системные требования

Sublime Merge работает на:
- **Windows** — 7 и выше
- **macOS** — 10.7 и выше
- **Linux** — большинство дистрибутивов (Ubuntu, Debian, Fedora и т.д.)

Требования к памяти минимальны — программа легко запускается даже на старых компьютерах.

### Загрузка и установка

Официальный сайт: **https://www.sublimemerge.com/**

На сайте вы найдёте установщики для всех платформ:

**macOS:**
```bash
# Через Homebrew
brew install sublime-merge

# Или скачайте dmg с официального сайта
```

**Ubuntu/Debian:**
```bash
# Через репозиторий Sublime
curl https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -
echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list
sudo apt-get update
sudo apt-get install sublime-merge
```

**Windows:**
Скачайте установщик с sublimemerge.com и следуйте инструкциям.

### Цена и лицензирование

Sublime Merge предоставляется по следующей модели:

- **Бесплатная пробная версия** — полная функциональность, без каких-либо ограничений. Время от времени появляется напоминание о покупке, но оно не мешает работе.
- **Одноразовая покупка** — $99 (один раз, на все устройства с одной учётной записью)
- **Подписка** — $75 в год (для получения обновлений и новых функций)

Пробная версия абсолютно полнофункциональна — вы можете использовать все возможности Sublime Merge без покупки. Это отличается от многих других инструментов, которые ограничивают функциональность в пробном периоде.

## Основной интерфейс

Когда вы открываете Sublime Merge в репозитории, вы видите три основные панели:

```
┌────────────────────────────────────────────────────────────┐
│ ☰ | Sublime Merge                                     🔍 ⚙│
├─────────────────┬──────────────────────────────────────────┤
│ Branches        │ Commit Graph                             │
│ ─ main          │                                          │
│ ─ develop       │ [●] feat: add auth                       │
│ ─ feature/auth  │  │ [●] fix: validation bug               │
│                 │  └─[●] merge main                        │
│ Tags            │                                          │
│ ─ v1.0.0        │                                          │
│ ─ v1.0.1        │                                          │
│                 │                                          │
│ Stash           │                                          │
├─────────────────┼──────────────────────────────────────────┤
│ Diff Viewer / File Changes                                 │
└────────────────────────────────────────────────────────────┘
```

### Левая панель (боковая)

Левая панель содержит:
- **Branches** — все локальные и удалённые ветки для быстрого переключения
- **Tags** — все метки (теги) в репозитории
- **Stash** — сохранённые изменения, которые можно восстановить позже
- **Remotes** — список удалённых репозиториев (origin, upstream и т.д.)

### Центральная панель (граф коммитов)

Это основная панель, которая показывает красивую визуализацию истории коммитов в виде графа. Каждый коммит представлен кружочком, а связи между ними показаны линиями. Это позволяет сразу понять структуру разработки и увидеть все ветки в контексте.

### Нижняя панель (просмотр изменений)

Здесь отображается содержимое выбранного коммита или diff между коммитами. Sublime Merge показывает изменения с синтаксисом, который вы используете в вашем коде.

## Особенности интерфейса Sublime Merge

### Синтаксис в diff

Одна из лучших особенностей Sublime Merge — это синтаксическое подсвечивание в diff просмотрщике. Когда вы смотрите на изменения в файле Python, HTML или JavaScript, код будет подсвечиваться правильно, что значительно облегчает чтение.

### Поиск и быстрая навигация

Используйте **Ctrl+P** (macOS: **Cmd+P**) для открытия палитры команд:

```bash
# Поиск коммитов по сообщению
: feat: add

# Поиск коммитов по автору
author: john

# Поиск файлов в коммитах
file: main.py
```

### Трёхпанельный редактор конфликтов

При возникновении конфликтов слияния Sublime Merge показывает три панели:
1. **Слева** — их версия (ветка, в которую вы объединяете)
2. **В центре** — результат слияния (файл, который будет создан)
3. **Справа** — ваша версия (текущая ветка)

Это делает разрешение конфликтов намного более интуитивным, чем в других инструментах.

### Встроенная поддержка SSH

Sublime Merge автоматически использует ваши SSH ключи, поэтому вам не нужно вводить пароль при push/pull операциях. Это работает как из коробки, если у вас правильно настроены SSH ключи в системе.

## Основные операции

### Клонирование репозитория

Используйте **Ctrl+Shift+P** (macOS: **Cmd+Shift+P**) и найдите "Clone Repository":

```bash
# Введите URL репозитория
https://github.com/username/repo.git

# Выберите директорию для сохранения
```

### Создание новой ветки

1. Откройте палитру команд (**Ctrl+Shift+P**)
2. Введите "Create Branch"
3. Введите имя новой ветки
4. Выберите базовую ветку (обычно главная)

Или щёлкните правой кнопкой на ветке в боковой панели и выберите "Create Branch".

### Создание коммита

После внесения изменений в файлы:

1. Откройте вкладку "Changes" в боковой панели
2. Выберите файлы, которые хотите добавить (они будут отмечены галочкой)
3. Введите сообщение коммита в поле вверху
4. Нажмите Enter или используйте кнопку "Commit"

```bash
# Пример сообщения коммита
feat: add user authentication

- Add login form
- Add password validation
- Add session management

Closes #42
```

### Интерактивный staging

Одна из мощных особенностей Sublime Merge — возможность выбирать конкретные строки кода для staging, а не целые файлы. Это полезно, когда вы сделали несколько разных изменений в одном файле и хотите их разделить на разные коммиты.

Чтобы добавить в staging конкретные строки:

1. Откройте diff для файла
2. Щёлкните на левой части конкретных строк, которые хотите добавить
3. Нажмите "Stage" или используйте контекстное меню

### Merge и Rebase

Для объединения ветки с текущей веткой:

1. Откройте палитру команд
2. Введите "Merge Branch"
3. Выберите ветку для объединения

Для rebase:

1. Откройте палитру команд
2. Введите "Rebase onto"
3. Выберите целевую ветку

## Горячие клавиши

Sublime Merge имеет множество полезных горячих клавиш:

```
Ctrl+P или Cmd+P          Палита команд
Ctrl+G или Cmd+G          Перейти к коммиту
Ctrl+Shift+P или Cmd+Shift+P  Открыть репозиторий
Ctrl+1, Ctrl+2, Ctrl+3    Переключиться между панелями
Enter                     Показать детали коммита
Space                     Развернуть/свернуть diff
h                         История для текущего файла
g                         Показать аннотации (git blame)
```

Все клавиши можно просмотреть и изменить в меню **Preferences → Key Bindings**.

## Сравнение с конкурентами

### Sublime Merge vs {{< relref "lazygit" >}}

| Характеристика | Sublime Merge | lazygit |
|---|---|---|
| **Цена** | Платная ($99 одноразово) | Бесплатна |
| **Интерфейс** | GUI | TUI (терминал) |
| **Скорость** | Очень быстро | Очень быстро |
| **Интерактивный rebase** | Хороший | Отличный |
| **Diff просмотр** | Лучший | Хороший |
| **Кривая обучения** | Средняя | Крутая |

### Sublime Merge vs GitHub Desktop

GitHub Desktop совершенно бесплатен и отличная программа для начинающих, но Sublime Merge быстрее и имеет лучший diff просмотр. GitHub Desktop лучше интегрирован с GitHub специфично.

### Sublime Merge vs GitKraken

GitKraken красивый и мощный, но тяжелее (на Electron) и дороже ($99 в год). Sublime Merge быстрее благодаря использованию нативного кода.

### Sublime Merge vs {{< relref "git-vscode" >}}

Git интеграция в VS Code хороша для быстрых операций, но Sublime Merge мощнее для серьёзной работы с Git. Если вы не используете VS Code, Sublime Merge — лучший выбор.

## FAQ

**Вопрос: Работает ли Sublime Merge без интернета?**
Да, полностью. Все операции локальны. Интернет требуется только для push/pull операций с удалённым репозиторием.

**Вопрос: Можно ли использовать одну лицензию на несколько устройств?**
Да, одна лицензия работает на все устройства, где вы авторизованы под одной учётной записью Sublime HQ.

**Вопрос: Есть ли плагины/расширения для Sublime Merge?**
Нет, Sublime Merge не имеет официальной системы плагинов, как Sublime Text. Однако разработчики постоянно добавляют новые функции в основное приложение.

**Вопрос: Как настроить внешний diff-инструмент?**
В настройках найдите "External Merge Tool" и укажите путь к вашему любимому инструменту (например, Beyond Compare или Meld). Затем используйте палитру команд "Diff with External Tool".

**Вопрос: Какой путь выбрать между Sublime Merge и {{< relref "github-desktop" >}}?**
Если вы работаете в основном с GitHub и хотите бесплатный инструмент, {{< relref "github-desktop" >}} хороший выбор. Если вам нужна максимальная скорость и лучший diff просмотр, Sublime Merge стоит инвестиции.

## Интеграция с другими инструментами

### Интеграция с Sublime Text

Если вы уже используете Sublime Text, вы можете добавить команду для открытия текущего файла в Sublime Merge:

```json
// Sublime Text меню Tools → New Snippet
{
  "command": "exec",
  "args": {
    "cmd": ["smerge", "$file"]
  }
}
```

### Интеграция с командной строкой

Добавьте Sublime Merge в PATH, чтобы открывать репозитории из терминала:

```bash
# macOS
smerge .

# Linux (если установлен через apt)
smerge .

# Windows (если установлен через scoop)
smerge .
```

## Заключение

Sublime Merge — это отличный выбор для разработчиков, которые ценят скорость, надёжность и качество инструментов. Хотя он не бесплатен, пробная версия полнофункциональна, и цена вполне разумна для инструмента, который вы будете использовать каждый день.

Основные преимущества Sublime Merge:
- Потрясающая скорость работы с большими репозиториями
- Лучший diff-просмотр среди всех GUI клиентов
- Простой и интуитивный интерфейс
- Полная поддержка SSH без дополнительной настройки
- Хороший интерактивный rebase и редактор конфликтов

Если вы ищете быстрый и надёжный Git клиент и не возражаете против небольшой платы, Sublime Merge идеальный выбор. Для бесплатных альтернатив рассмотрите {{< relref "github-desktop" >}} или {{< relref "lazygit" >}}.
